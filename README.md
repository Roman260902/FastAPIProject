# FastAPIProject

## Запуск проекта

Запуск скрипта run.bat для пользователей Windows

Запуск скрипта run.sh для пользователей Unix

Скрипт создает виртуальное окружение, устанавливает библиотеки

После этого запуститься бэкенд и фронтэнд на стримлите

## Загрузка данных

В папке testing находиться csv, сгенерированный на основе выгрузки операций из Т-Банка

На первой вкладке загружаем операции из файла.

На второй вкладке есть возможность добавить операцию самостоятельно.

На третей вкладке аналитика трат по месяцу.
